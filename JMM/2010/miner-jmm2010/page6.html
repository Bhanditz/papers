Math on the Web formats in the Wild

  • Images with "TeX" code (driven by blogs, wikis, tools, etc.)
    • Alt tags (Wikipedia), comments (Planet Math), URLs (Google Docs), image comments (MathType)
  • Some content is in text
    • HTML Math, TeX source (MathSciNet), ASCII Art (MathForum)
  • Some is in the DOM
    • MathML (Firefox), <span>s and CSS (MathJax)
