Long Term Strategy: Reduce Balkanization

  • MathJax establishes a defacto TeX-based input language standard
  • Reduces the problem space to TeX and/or MathML
  • Via extensible input Jax architecture, more legacy formats could be supported over time, e.g. ASCIIMathML
  • Provides a stable interface to a changing backend
    • Smooth transition to browser MathML support with native cut and paste
